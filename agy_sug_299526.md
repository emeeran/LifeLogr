## 🚀 Core Consolidation Blueprint (Consolidate)

To execute the pruning successfully, we must follow a precise sequence that keeps the application stable and preserves the core journaling experience:

### Step 1: Update Pydantic Schemas and Database Registries
1. Remove deleted models from `backend/app/models/__init__.py`.
2. Clean up foreign keys in `Entry` and `Media` ORM classes (remove relationships to `EntryEmbedding`, `EntrySentiment`, `EntryPrompt`, and `OCRResult`).
3. Re-run database migrations or drop the local database to clean up the schema since this is an offline-first, local-first app.

### Step 2: Redirect Search Service to SQLite FTS5 (Native & Ultra-Fast)
Currently, `SearchService` executes slow, un-indexed vector cosine similarity checks in Python RAM:
```python
# app/services/search_service.py
# BEFORE: Loads every embedding from SQLite and does cosine checking in Python
for row in rows:
    vec = json.loads(row.embedding)
    score = cosine_similarity(query_vec, vec)
```

**After Pruning:** We rely solely on SQLite's native FTS5 (Full-Text Search) virtual table index. SQLite FTS5 compiles to native C code, uses zero external dependencies, performs keyword matching in **< 5ms**, and does not freeze the async event loop.

### Step 3: Streamline the Entry Service
Remove background tasks trigger from `EntryService`:
```diff
# app/services/entry_service.py
- from app.services.enrichment_service import EnrichmentService
...
async def create(self, data: EntryCreate) -> Entry:
    ...
-   # Trigger background AI enrichment
-   EnrichmentService.schedule(entry.id, entry.title, entry.body)
```

### Step 4: Move Text-to-Speech (TTS) entirely to the Web Browser (Frontend)
Instead of streaming speech audio via Microsoft's cloud endpoints in the backend (which breaks offline-first requirements), we utilize the standard Web Speech API directly in Vue 3:

```javascript
// frontend/src/composables/useSpeech.ts
export function useSpeech() {
  const speak = (text: string, voiceName: string = "Google US English") => {
    if (!window.speechSynthesis) return;
    
    // Cancel any active speech
    window.speechSynthesis.cancel();
    
    const utterance = new SpeechSynthesisUtterance(text);
    const voices = window.speechSynthesis.getVoices();
    utterance.voice = voices.find(v => v.name === voiceName) || null;
    
    window.speechSynthesis.speak(utterance);
  };
  
  const stop = () => {
    window.speechSynthesis?.cancel();
  };

  return { speak, stop };
}
```
* **Performance Gain:** 0ms backend delay, 0 bytes of network traffic, fully offline.

### Step 5: Clean Up Frontend App Shell
1. Remove AI summary panel drawer, plugin tab, reminders configuration list, and templates setup screen.
2. Polish the timeline, chronological list, and search views to focus solely on the rich markdown writing interface, media grids (images, videos, documents), and voice note players.
